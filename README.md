# Max Verstappen Fan Project 🏎️

Это мой учебный проект — фан-сайт, посвященный трехкратному чемпиону Формулы-1 Максу Ферстаппену.

## 🛠 Технологии
* **HTML5** — структура сайта.
* **CSS3** — стилизация (Flexbox, Google Fonts).
* **JavaScript** — в планах добавить интерактив.

## 🚀 Функционал
- [x] Главная страница с биографией.
- [x] Список главных достижений с кастомными маркерами.
- [x] Карточка пилота в фирменных цветах Red Bull.
- [in progress] Мобильная адаптация.

## Как запустить
Просто открой файл `index.html` в любом современном браузере.
